# Student Registration Form

A simple and responsive **Student Registration Form** built using **HTML5**. This project demonstrates the use of basic HTML form elements, input types, and table creation to collect and display student information.

## Features

- Student registration form
- Full Name input field
- Email input field
- Password input field
- Gender selection using radio buttons
- Skills selection using checkboxes
- Date of Birth picker
- Register button
- Submitted Students section
- Dummy student data displayed in a table
- Beginner-friendly HTML structure

## Technologies Used

- HTML5

## HTML Concepts Covered

- HTML Document Structure
- Headings
- Forms
- Labels
- Input Elements
- Input Types
  - Text
  - Email
  - Password
  - Radio
  - Checkbox
  - Date
- Buttons
- Tables
- Table Rows (`<tr>`)
- Table Headers (`<th>`)
- Table Data (`<td>`)
- Line Breaks (`<br>`)

## How to Run

1. Clone this repository.
2. Open the project folder.
3. Open `index.html` in any web browser.

No additional software or installation is required.
